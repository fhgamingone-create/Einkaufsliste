EINKAUFSLISTE APP - GANG 1 BIS 10
=================================

Diese Version ist für Handy-Nutzung optimiert und sieht mehr wie eine App aus.

Dateien:
- index.html
- style.css
- script.js
- manifest.webmanifest
- service-worker.js
- icon.svg

So lädst du es auf GitHub hoch:
1. Entpacke die ZIP-Datei.
2. Lade ALLE Dateien in den Hauptordner deines GitHub-Repositories hoch.
3. Achte darauf, dass index.html direkt im Hauptordner liegt.
4. Gehe in GitHub zu Settings > Pages.
5. Wähle Deploy from a branch.
6. Branch: main
7. Folder: /root
8. Speichern.

Funktionen:
- Handyfreundlicher App-Look
- Gang 1 bis 10 direkt auswählbar
- Sortierung nach Gang
- Sortierung alphabetisch A-Z
- Artikel abhaken
- Artikel bearbeiten
- Artikel löschen
- Erledigte Artikel löschen
- Lokale Speicherung im Browser
- Export und Import als JSON
- PWA-Dateien enthalten: kann je nach Browser als App zum Home-Bildschirm hinzugefügt werden

Wichtig:
Die Daten werden weiterhin lokal im Browser gespeichert.
Das heißt: Auf einem anderen Handy oder Laptop ist die Liste nicht automatisch synchronisiert.
Dafür bräuchte man später Firebase, Supabase oder eine andere Online-Datenbank.
